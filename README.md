# README

### Project Setup
- Clone the project:

        git clone git@github.com:TahirAhmad01/CourseManagementAplication-ROR.git
        cd CourseManagementAplication-ROR
    
- Install dependencies:

        bundle install
        yarn install
        rake db:create
        rake db:migrate
        rake db:seed
        
- Run the Rails server:

        rails s
